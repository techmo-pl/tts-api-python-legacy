# ⚠️ Deprecated

This repository is archived and no longer maintained.

It has been superseded by two separate packages following the ASR convention:

| Purpose | Repository | PyPI package |
|---------|-----------|--------------|
| gRPC stubs (proto → Python) | [techmo-pl/tts-api-python](https://github.com/techmo-pl/tts-api-python) | `tts-api` |
| Python client | [techmo-pl/tts-client-python](https://github.com/techmo-pl/tts-client-python) | `tts-client` |

## Migration

Replace any dependency on this package with `tts-api`:

```sh
pip install tts-api
```
